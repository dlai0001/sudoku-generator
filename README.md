> Sudoku generator for node


## Install

```sh
$ npm install --save sudoku-generator
```


## Usage

```js
var sudokuGenerator = require('sudoku-generator');

sudokuGenerator('Rainbow');
```

```sh
$ npm install --global sudoku-generator
$ sudoku-generator --help
```
